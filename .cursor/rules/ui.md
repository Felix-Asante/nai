# UI library

Use Shadcn component documentation from:
https://ui.shadcn.com/llms.txt

Always import UI components from Shadcn.
Prefer Shadcn components instead of plain web primitives.

# Cursor rules — styling

Use Tailwind classes for styling components
Do not use inline styles if an equivalent Tailwind utility exists
Prefer utility and mobile first approach from Tailwindcss.
